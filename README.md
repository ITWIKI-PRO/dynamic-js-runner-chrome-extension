# Dynamic JS Runner (Chrome Extension)

Расширение позволяет запускать произвольный JavaScript на страницах, которые вы посещаете, по заданным вами правилам (match patterns).

## Установка (Developer mode)
1. Откройте `chrome://extensions/`
2. Включите **Developer mode**
3. Нажмите **Load unpacked** и выберите папку расширения

## Использование
- Откройте страницу **Правила**: клик по кнопке «Правила» в попапе или через Chrome Extensions → Details → Extension options.
- Добавьте правило:
  - **Шаблон URL** (Chrome match pattern), например `*://*.example.com/*`
  - **Код** — любой JS (исполняется в `async () => { ... }`), ошибки выводятся в консоль страницы.
  - **World**:
    - `MAIN` — код выполняется в контексте страницы (может видеть JS-переменные страницы)
    - `ISOLATED` — изолированный мир контент-скрипта
- Скрипт запускается автоматически при полной загрузке страницы (status=complete). Кнопка **Запустить сейчас** выполняет подходящие правила принудительно.

## Пример скрипта
```js
// Удалить все .mb, где нет span.vrico (пример)
document.querySelectorAll('.mb').forEach(el => {
  if (!el.querySelector('span.vrico')) el.remove();
});
```

## Важно про безопасность
Расширение исполняет произвольный код на сайтах, которые вы посещаете.
- Используйте только собственные скрипты, которым доверяете.
- Не импортируйте правила из неизвестных источников.


## RegExp для URL
Теперь в правилах есть выбор типа URL-условия: **Chrome match pattern** или **RegExp**.
- RegExp можно вводить как обычный шаблон (`^https?://...$`), либо как литерал `/.../i`.
